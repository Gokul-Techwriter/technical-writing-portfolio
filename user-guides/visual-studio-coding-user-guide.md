## Introduction

Visual Studio Code (VS Code) is a code editor developed by Microsoft. This guide helps users navigate the VS Code interface, including the Explorer, Extensions, Terminal, and Debug features on Windows.

## How to Use This Guide

Read this guide from top to bottom if you are a beginner. If you are looking for a specific feature, use Ctrl+F to search by keyword or jump to the relevant section using the Table of Contents.

## Prerequisites

Before you begin, ensure the following requirements are met:

- [ ]  Windows 10 or later

- [ ] Minimum 4GB RAM

- [ ] ARM64 or x64 processor; Quad-core or better

- [ ] 850 MB of disk space

- [ ] Display resolution of 1366 x 768 or higher

- [ ] VS Code is installed on your device.  
Download: <https://code.visualstudio.com/download>

## Key Features

### Feature 1: Integrated Terminal

The Integrated Terminal allows users to run commands directly within VS Code without switching to an external application.

### Feature 2: IntelliSense

IntelliSense automatically suggests code completions as users type, eliminating the need to memorize syntax or commands.

### Feature 3: Extensions Marketplace

Extensions Marketplace allows users to install additional tools and add-ons directly within VS Code, eliminating the need to access them externally.

### Feature 4: Git Integration

Git Integration allows users to track code changes, commit, and manage version control directly within VS Code without switching to an external application.

## Troubleshooting

### Issue 1: Installation fails

**Cause:**

- The system doesn't have enough storage for the installation.

- Corrupted installer file

- The system doesn't have the minimum requirements to complete the installation.

**Fix:**

1. Ensure that your system has sufficient storage for the installation. If not, remove unnecessary files.

2. Verify that your system meets the minimum requirements to run the app.

3. Download the latest setup file from the official page of VS Code.

4. Attempt the action of installing it again.

5. Download the VS Code Zip Archive and run it if the step above fails.

**Expected Result:** Visual Studio Code is installed successfully without any errors.

### Issue 2: Extensions Not Installing

**Cause:**

- Unstable internet connection

- Extensions permission denied

- Outdated Version of VS Code

- Insufficient space

**Fix:**

1. Ensure that your system has a stable internet connection.
 
2. Click on the **Settings icon** > **Settings** (**Ctrl+,**) > **Extension Settings**; make sure the type of extension you prefer to install is allowed. If not, allow it.

3. Update VS Code by clicking **Help** > **Check for Updates** and installing any available updates.
 
4. Ensure that your system has sufficient storage for the installation.
 
5. Retry the attempt to install the extension.

**Expected Result:** Extension is installed successfully without any errors.

### Issue 3: Terminal Not Opening

**Cause:**

- No workspace or folder open
- Newly installed/updated extensions are blocking.
- Corrupted Terminal settings

**Fix:**

1. Ensure you have created a workspace or folder. If not, create one.

2. Check whether any newly installed or updated extension is blocking. If yes, uninstall those extensions.

3. Reset the Terminal settings by pressing **Ctrl+Shift+P**, typing **Reset Settings**, and pressing **Enter** to restore default settings.

**Expected Result:** Terminal opens successfully without any errors.

## FAQ

**Q1: What is VS Code?**

**A:** Visual Studio Code (VS Code) is a code editor 
made by Microsoft. It offers features like the Integrated Terminal, IntelliSense, Extensions Marketplace, and Git Integration.

**Q2: Which Operating Systems (OS) support VS Code?**

**A:** The following OSs support VS Code:

- Windows 10 or later (64-bit)

- macOS: macOS 10.15 or later

- Linux: Ubuntu 20.04 or later, Debian 10 or later, Red Hat Enterprise Linux 8 or later, Fedora 36 or later

**Q3: Does VS Code need an internet connection?**

**A:** VS Code requires an internet connection for features like installing extensions, automatic updates, and Git integration. Core editing works offline.

**Q4: What is IntelliSense?**

**A:** IntelliSense is a built-in feature in VS Code which suggests code completions as you type. This helps you write code faster, eliminating the need to memorize every command or syntax.

**Q5: Why does the app get closed in the middle of use?**

**A:** This issue may occur due to an outdated version; to resolve it, do the following:

1. Click **Help** at the top of your screen.

2. Select **Check for Updates**

3. If there is an update available, do it.
