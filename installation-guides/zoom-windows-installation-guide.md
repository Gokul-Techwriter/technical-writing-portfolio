## 1. Introduction

### 1.1 Purpose

This guide explains how to install and use the Zoom app on Windows.

### 1.2 Scope

The guide covers:

- System requirements
- Installation procedure
- Verification
- Uninstalling procedure

### 1.3 Intended Audience

This guide is intended for:

- End users
- System administrators
- IT support teams

---

## 2. Overview 

Zoom is a video conferencing app. It’s great for online meetings, video calls, webinars, and screen sharing. It allows people to talk in real-time from anywhere on the internet.

## 3. System Requirements

Before you begin, ensure the following requirements are met:   

### 3.1 Software Requirements

- Stable internet connection  
- Windows 10 or later

### 3.2 Hardware Requirements

- Minimum 4 GB of RAM
- Dual-core 2 GHz or higher  
- 64-bit processor  
- 250 MB disk space

<small/>**Note:** Requirements may change based on Zoom updates. Refer to the official Zoom website for the latest specifications.</small>

## 4. Pre-Installation Checklist 

Before installing Zoom, ensure your desktop has the following requirements:

- Internet Connection  
- Administrator Access
- The system meets the minimum requirements

## 5. Installation Procedure

### 5.1 Download Zoom

1. Open a web browser.  
2. Go to the official **Zoom website**.

<img src="../../../../../12/z/image_page2_1_X13.png" width="40%" alt="1" />

3. Select the **installer** that matches your system architecture:

- 64-bit for most modern computers.  
- 32-bit for legacy systems.  
- ARM64 for ARM-based devices.

<img src="../../../../../12/z/image_page2_1_X135.png" width="40%" alt="1" />

4. Click **Download** after selecting the **installer**.

**Result:** Zoom installer download completed.

### 5.2 Install Zoom

1. Open the **Downloads folder**.

<img src="../../../../../12/z/image_page3_3_X19.png" width="40%" alt="1" />

2. Double-click the downloaded **Zoom installer file** and wait for the installing to complete.

<img src="../../../../../12/z/image_page3_4_X20.png" width="40%" alt="1" />

**Result:** Zoom is installed without any errors.

## 8. Uninstalling Zoom

To uninstall the Zoom app:

1. Open **Control Panel**. 

<img src="../../../../../12/z/image_page5_1_X30.png" width="40%" alt="2" />

2. Go to **Programs** > **Uninstall a Program**

<img src="../../../../../12/z/image_page5_2_X31.png" width="40%" alt="2" />

3. Select **Zoom Workplace**.  

<img src="../../../../../12/z/image_page5_3_X32.png" width="50%" alt="2" />

4. Click **Uninstall** and wait for the uninstalling to complete.

<img src="../../../../../12/z/image_page5_4_X33.png" width="40%" alt="2" />

**Result:** Zoom uninstalled.

## 9. Troubleshooting

**Problem 1:** Download fails  
**Cause:** Unstable internet connection.  

**Solution:** 

1. Click **Start** > **Settings** > **Network & Internet**  
2. Ensure your network connection is working properly.  
3. Restart the download

**Problem 2:** App not working properly  
**Cause:** Incorrect version.  

**Solution:** 

1. Click **Start** > **Settings** > **System** > **About**  
2. Check your system specifications.  
3. Re-download the **installer** according to the system specifications. 
4. Relaunch the app.
