## Introduction

Postman is an API platform developed by Postman, Inc. This guide helps users to successfully download, install, verify, and uninstall the app.

## Who This Guide Is For

This guide is intended for:

- Software Developers and IT Support Teams looking to install Postman on their Windows for the first time.

- Beginners with basic computer knowledge looking to install Postman on their Windows.

## Downloading Postman

### Step 1: Visit the official website

Go to the Official website of Postman: https://www.postman.com/downloads/

> ⚠️ **Warning:** Always download from the official 
> website only. Avoid third-party download sites.

### Step 2: Download the app 

Click on **Windows x64** to download the installer file and wait for its completion

> 💡 **Tip:** Note where the file is downloaded -
> you will need it in the next step.

**Expected Result:**  The installer file is downloaded successfully without any error

---

## Installation Steps

### Step 1: Find the downloaded file

Go to the Downloads folder in your Windows.

### Step 2: Install the app

Double-click Postman (x64) to start the installation process; Postman installs automatically.

### Step 3: Launch Postman

Postman opens automatically once installation is complete.

**Expected Result:** Postman is installed successfully without any errors.

---

## Post-Installation Setup

### Step 1: Create a free account

1. Enter your email address, and click **Create Free Account**.

2. Set a name and password for your account; click **Create Free Account**.

3. Verify the email by using the code sent to it.

4. Click **Allow access** in the **window security alert tab**.

5. Enter your name and role for the workspace; write the request in the **request box**.

6. Click **Get started with AI**.

> 💡 **Tip:** Enter “Sent a GET request” in the request box for a start if you are confused.

**Expected Result:** Account is created successfully without any errors.

### Step 2: Create a new workspace

1. Click on the **drop-down icon** on the top left.

2. Choose **Create**.

3. Tick according to your preference; click **Next**.

4. Enter the name for your workspace > select the **Workspace type**; choose the type of **Manage access**.

5. Click **Create**.

**Expected Result:** New workspace is created successfully.

### Step 3: Change the Color Theme

**Steps:**

1. Click on the **Settings icon** > **App settings**.

2. Choose **Themes**

3. Select the theme for daytime.

4. Select the theme for nighttime.

> 💡 **Tip:** Changing the color theme to dark is easier on the eyes during long sessions.

**Expected Result:** Color theme is changed successfully.

## Uninstallation

### Windows

1. Open **Start menu** > search **Control Panel** > click **Open**

2. Click on **Programs** > **Programs and Features**

3. Search **Postman x64 12.2.2** 

4. Click on the **Postman x64 12.2.2** > Choose **Yes** for the uninstall confirmation pop-up.

5. Wait for the uninstallation to get completed > Click **Ok** for the confirmation 

**Expected Result:** Postman is uninstalled from Windows without any errors.

## Troubleshooting

### Issue 1: Installing fails

**Cause:**  

- Insufficient storage space to install Postman.

- System doesn't have the minimum requirements.

- Corrupted installer file. 

**Fix:**  

1. Ensure there is enough storage space for the installation.

2. Confirm your system has the minimum requirements to install the app: Open **Start menu** > **Settings** > **System** > **About**.

3. Ensure that the installer file is of the latest version. if not, download the latest version from the official website.

4. Attempt the action again. 

> ⚠️ **Warning:** Do not interrupt the installation 
> process once it has started.

**Expected Result:** Postman is successfully installed on Windows.

### Issue 2: App closing automatically

**Cause:**

- Outdated version of the app.

- Corrupted installer file.

**Fix:**

1. Update the app: **Menu** > **Help** > **Check for Updates** > **Update**.

2. Download the latest installer file from the official website and install it.

**Expected Result:** The automatically closing issue is resolved successfully.

### Issue 3: Unable to Uninstall

**Cause:**

- Postman is still running in the background.

- Insufficient administrator permissions.

**Fix:**

1. Open **Task Manager** (**Ctrl+Shift+Esc**) and end any running Postman processes.

2. Ensure you have administrator access on Windows.

3. Restart the system.

4. Attempt the uninstallation again.

**Expected Result:** Postman is uninstalled successfully from Windows.

## FAQ

**Q1: Is the Postman app available on any mobile device?**
**A:** No, the Postman app is not officially available on mobile devices, but you can access the web version on mobile.

**Q2: Can I manually update the Postman app?**
**A:** Yes, for updating the app manually, click on the **Menu icon** (**Three lines**) > **Help** > **Check for Updates** > if a new update is available, click **Update**.

**Q3: The app is not opening. What should I do?**
**A:** This may happen due to the insufficient system requirements/corrupted installer file. Confirm your system meets the minimum requirements and re-download the latest installer file from the official website.
