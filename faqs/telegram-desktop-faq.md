## 4. General FAQs

**Q1. What is Telegram Desktop?**

**A:** Telegram Desktop is the official desktop version of the messaging app called Telegram that allows users to send text messages, voice messages, photos, videos, files, and make voice/video calls over the internet.

**Q2. Which operating systems are supported for using Telegram Desktop?**

**A:** Operating systems that support Telegram Desktop:

- Windows (Windows 7 or later)
- macOS (macOS 10.11 (El Capitan) or later)
- Linux (64-bit Linux distribution (x86_64) with glibc 2.17 or later)

## 5. Installation FAQs

**Q1. How do I install the application?**

**A:**
1. Go to the official website of Telegram.
2. Download the installer file according to your system (Windows/macOS/Linux).
3. Run the installer file to install Telegram Desktop on your system.

**Q2. Why does installation fail?**

**A:** Installation may fail due to insufficient disk space, a corrupted installer file, or an outdated operating system.

## 6. Account & Login FAQs

**Q1. How to log in to Telegram Desktop?**

**A:**
1. Select the country and enter your registered phone number.
2. Verify the number using the code sent to your Telegram app on mobile (or you can directly log in through scanning the QR code displayed on the desktop with your mobile app)

**Q2. Why am I unable to log in?**

**A:** Ensure a stable internet connection and verify that the entered credentials for login are correct.

## 7. Feature FAQs

**Q1. How to archive a chat?**

**A:**
1. Right-click on the chat you want to archive
2. Click Archive

**Q2. How to change chat theme?**

**A:**
1. Click the Menu button (≡) on the top left corner
2. Choose Settings > Chat Settings; then set the chat themes you want

## 8. Troubleshooting FAQs

**Q1. Telegram Desktop fails to update. Why?**

**A:** Update failure occurs due to insufficient disk space. Ensure a stable internet connection > confirm there is enough disk space for the update; if not, clear some space.

**Q2. Why is the application running slowly?**

**A:** Slow performance may occur due to low system resources or outdated software versions. Ensure a stable internet connection and confirm your system meets the basic requirements to run the app.

## 9. Security & Privacy FAQs

**Q1. Is the user data of Telegram Desktop encrypted?**

**A:** Yes, Telegram Desktop uses end-to-end encryption for secret Chats, while regular cloud chats are encrypted in transit and on Telegram's servers.

**Q2. How is personal information protected?**

**A:** Telegram protects personal information through end-to-end encryption in secret chats, secure cloud storage, two-step verification, privacy controls, and minimal data collection.
